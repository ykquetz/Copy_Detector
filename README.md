# Copy_Detector
CustomTkinter ile geliştirilmiş, disk alanınızı optimize eden modern bir kopya dosya bulucu. Dosyaları önce boyutuna, ardından SHA-256 hash değerlerine göre tarayarak birebir kopyaları tespit eder. Görsel önizleme, orijinal dosyayı koruyarak toplu silme ve HTML raporlama özellikleriyle hızlı ve güvenli bir temizlik deneyimi sunar.
